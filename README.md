# AWS Diagram Interpreter 🚀🤖

Welcome to Chainlit AWS Diagram Interpreter, your go-to tool for making sense of AWS architecture diagrams and generating CloudFormation code with ease!

## Overview ℹ️

AWS Diagram Interpreter is designed to simplify the process of understanding and working with AWS architecture diagrams. By uploading your AWS diagram image, this bot will provide a clear explanation of the architecture and generate corresponding CloudFormation  code, streamlining your AWS infrastructure deployment process.

## Features ✨

- **AWS Diagram Interpretation:** Upload your AWS architecture diagram image, and let the bot decipher it for you.
- **CloudFormation Code Generation:** Receive ready-to-use CloudFormation code based on the interpreted architecture.
- **User-Friendly Interface:** Intuitive interface for seamless interaction and effortless AWS architecture understanding.

## Usage 🛠️

1. **Upload Diagram:** Submit your AWS architecture diagram image.
2. **Interpretation:** Receive a detailed explanation of the architecture.
3. **Code Generation:** Get CloudFormation code for easy deployment.

## Get Started 🚀

To get started with Chainlit AWS Diagram Interpreter, simply follow these steps:

1. Clone this repository.
2. Install the required dependencies (requirements.txt).
3. Create a .env file
4. Run the application(chainlit run app.py) locally or deploy it to your preferred hosting platform.

## Technologies Used 🛠️

- **Python:** Backend logic and server-side operations.
- **Chainlit:** Interactive front-end interface and application flow management.
- **Claude 3 LLM:** Leveraged for natural language processing and interpretation of AWS architecture diagrams.
